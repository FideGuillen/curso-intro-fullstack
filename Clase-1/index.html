Aprendemos a usar html

Uso de los tags h1, h2, ..., h6 y p

Por defecto el tag h1 es para títulos, el resto para distintos tamaños de letras.
Los p son para párrafos.
En html debemos tener en cuenta que siempre debemos guardar cambios cada vez que modificamos el código o configurarlo desde el file->autosave

Uso de los ul u ol

El UL se usa cuando queremos listas desordenadas (no numeradas) como por ejemplo: lista de ingredientes en una receta Y el ol se usa para lista numeradas.
